## 1. Purpose

This register is the master list of all controlled documents in the WaterFilterCBZ quality
management system, per [SOP-001 Document and Record Control](SOP-001_Document_and_Record_Control.md).
It records, for each document, the current revision, status, owner, and review due date so that
only current, approved documents are in use (ISO 13485 §4.2.4).
